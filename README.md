# Lubuntu System Tools 🛠️

This repository contains custom Bash scripts I wrote to manage my Lubuntu workstation.

## Included Tools

### 🕵️ System Access Logger (`logger.sh`)
This script records a timestamp, the current user, and system uptime into a text file. I use this to track my development sessions.

**How to use:**
1. Give it permission: `chmod +x logger.sh`
2. Run it: `./logger.sh`
3. View the log: `cat ~/Documents/access_report.txt`
